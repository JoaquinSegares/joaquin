Para los titulos se utiliza comillas angulares y entre h1 y h6 depende del titulo. Es decir que h1 es mas importante que el h6, va en escala. Para parrafo se utiliza comillas angulares y una p en el medio. Este texto usa el strong para darle enfasis (bold).

Listas con ul y li:

  • ul es para unorderd list.
  • cada elemento esta en lista con el li.
  • la lista es compuesta por el ul y entre aquello se pone li para cada elemento en lista.

Aqui arriba se ve un elemento remplazable, es una imagen por la cual no ocupa una etiqueta de cierre. Para subir esta imagen se oucpa descargarla y depositarla en el folder de imagenes y despues poner el nombre en el src para implementarla en el sitio web. El alt es usado para describir la imagen por is acaso no quisiera cargar la imagen. El title (titulo) es usado para darle un titulo a la imagen para cuando el mouse del usuario este encima de la imagen, no tiene que ser descriptivo pero ayuda. Todo esto son los atributos. Height y Width cambian el tamaño de la imagen a como uno guste utilizarla. El atributo hidden esconde todo el elemento que uno enseña, es boleano. ID es el identificador, es usado para identidicar el elemento pero nunca se puede repetir con otros elementos, funciona como un id de la vida real, para identificar.
